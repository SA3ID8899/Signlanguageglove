This is a project to detect sign language using glove, the fourth
finger was neglected as it has low uses in our function.

To apply this code follow this steps;
1- Apply all the .h and .c files in ur project
2- Copy and paste the main .c file in your project with the
appropriate libraries as in the file
3- Check the threshold resistance for each flex sensor
individually and put the value read in the threshold 
variable for each resistance
4- Ctrl+S, Ctrl+B, Ctrl+Alt+U
